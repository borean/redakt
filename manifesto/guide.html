Download Redakt

No installation wizards. No dependencies. Download, open, and start de-identifying.

MAC Apple Silicon
MAC Intel
WINDOWS x64
LINUX x64 (.deb)

The app is ~8 MB. Choose from 3 model sizes (6 GB to 21 GB) — downloads automatically on first launch.
macOS: Open the DMG and drag Redakt to Applications.
Windows: Run the installer (.exe) and follow the prompts.
Linux: Install with sudo dpkg -i Redakt_*.deb. RPM also available on GitHub.
